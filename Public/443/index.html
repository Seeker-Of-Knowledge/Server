What is this?

You've found a third-party OpenSend Messaging server! OpenSend Messaging allows for private, encrypted messaging. To make OpenSend Messaging more decentralized, third-parties host messaging servers. You can connect to these servers via OpenSend Messaging by providing the domain you're currently at.

Can I host one?

Certainly! All the OpenSend Server code is open-source. You can view (and download) it via GitHub.
